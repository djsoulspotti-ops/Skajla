Crea un chatbot AI intelligente e personalizzato integrato nella piattaforma SKAILA. Il bot deve essere adattabile per ogni utente, con capacità di machine learning e deep learning per supporto educativo avanzato.
Core Features Richieste
Personalizzazione Utente:

Profilo Adattivo: Il bot impara stile di studio, difficoltà preferite, materie forti/deboli di ogni utente
Modalità Conversazione: Formale per professori, amichevole per studenti, professionale per dirigenti
Memoria Persistente: Ricorda conversazioni precedenti, progressi, preferenze personali
Customizzazione: Nome bot personalizzabile, avatar scelto dall'utente, tono di voce regolabile

Intelligenza Artificiale Avanzata:

NLP Engine: Comprensione linguaggio naturale italiano avanzata con riconoscimento contesto
Machine Learning: Algoritmi di apprendimento per migliorare risposte basate su feedback utente
Deep Learning: Reti neurali per analisi pattern di apprendimento e predizione difficoltà
Knowledge Base: Database educativo aggiornabile con curriculum italiano

Funzionalità Educative Specifiche:

Tutor Personalizzato: Spiegazioni adattate al livello dello studente con esempi mirati
Generazione Esercizi: Creazione automatica quiz/problemi basati su programma scolastico
Analisi Performance: Tracking progressi con insights e suggerimenti miglioramento
Supporto Homework: Aiuto compiti senza dare soluzioni dirette, guidando il ragionamento
Preparazione Verifiche: Simulazioni esami e ripasso mirato su lacune identificate

Integrazione Sistema:

API REST: Endpoints per tutte le funzionalità bot accessibili dal frontend
Real-time Chat: Integrazione WebSocket per conversazioni fluide
Database Learning: Storage pattern utente, conversazioni, performance analytics
Multi-modal: Supporto testo, immagini, voice-to-text per accessibilità completa

Specifiche Tecniche
Backend AI Architecture:

Framework: Python con TensorFlow/PyTorch per deep learning
NLP Library: spaCy o Transformers per processing linguaggio italiano
ML Pipeline: Scikit-learn per algoritmi machine learning classici
Vector Database: Per similarity search e retrieval knowledge base
API Gateway: FastAPI per gestione richieste bot con rate limiting

Frontend Integration:

Chat Interface: Widget chat moderno integrabile in qualsiasi pagina SKAILA
Voice Integration: Speech-to-text e text-to-speech per interazione vocale
Rich Media: Supporto rendering formule matematiche, grafici, diagrammi
Mobile Optimized: Interfaccia responsive per tablet/smartphone

Modelli AI da Implementare:

Sentiment Analysis: Per rilevare frustrazione/confusione studente
Intent Classification: Categorizzazione automatica richieste utente
Knowledge Graph: Mapping concetti e relazioni tra materie
Difficulty Assessment: Valutazione automatica livello difficoltà contenuti
Personalization Engine: Raccomandazioni contenuti basate su profilo utente

Privacy & Security:

Data Encryption: Crittografia conversazioni e dati personali utente
GDPR Compliance: Gestione consensi e diritto cancellazione dati
Content Filtering: Moderazione automatica contenuti inappropriati
Role-based Access: Diversi livelli accesso AI per studenti/professori/dirigenti

Training Data Requirements:

Dataset conversazioni educative italiane
Curriculum materie scolastiche per ogni anno
Esempi esercizi e soluzioni per training
Feedback loops per continuous learning

Expected Deliverables:

Chatbot Core Engine funzionante con ML/DL capabilities
Training Pipeline per aggiornamento continuo modelli
Admin Dashboard per monitoraggio performance e configurazione
API Documentation completa per integrazione
Demo Interface per testing funzionalità base
Analytics System per tracking engagement e effectiveness

Implementa un sistema AI completo, scalabile e pronto per ambiente educativo professionale con focus su personalizzazione avanzata e capacità di apprendimento continuo.
